# daign-handle

[![NPM package][npm]][npm-url]

Define drag actions for all DOM elements.

## Installation

```sh
npm install @daign/handle --save
```

## Scripts

#### Build

    npm run build

#### Run lint analysis

    npm run lint

#### Run unit tests with code coverage

    npm run test
